# Page variables

常用或重要的非常规**页面**变量合集\
它们可能来自某个扩展，也可能是自己写的

## Sheet

|变量|类型|默认值|用处|提供者|
|--|--|--|--|--|
|sitemap|boolean|true|添加至网站地图|jekyll-sitemap|
|noindex|boolean|false|声明禁止搜索引擎添加至索引|blog|
|creative_commons.ShortName|string|none|当前页面的 cc 协议名称|blog|
|creative_commons.URL|string|none|当前页面的 cc 协议的 url 链接|blog|
